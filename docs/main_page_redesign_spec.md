## 1. 디자인 컨셉

### 1.1 핵심 메시지

> **"전국 식당에 매일, 신선한 고기를 안정적으로."**

육류 B2B의 본질(**신선함 + 안정성 + 신뢰**)을 전면에 내세우고,
디지털/AI는 그 안정성을 뒷받침하는 **수단**으로 위치.

### 1.2 톤 매핑

| 축 | 비율 |
|----|------|
| 신뢰감 (B2B 정통) | **60%** |
| 혁신감 (데이터/AI) | **25%** |
| 프리미엄 (고급육류) | **15%** |

### 1.3 페르소나 우선순위

1. **개인 식당 사장** (메인 타깃) — "내일 영업할 고기 안정적으로 받자"
2. **중형 외식 체인 구매담당** — "여러 점포 발주 한번에"
3. **농가/도축장 공급사** — "안정적 판로"
4. **투자자/파트너** — 기존 LandingPage 일부 유지 (별도 경로 `/about` 또는 푸터 링크)

## 7. 기존 페이지 처리

**기존 LandingPage.tsx의 IT/투자자 톤 콘텐츠**는 어떻게 할 것인가?

옵션:
- **A. 전부 삭제** — 깔끔하지만 기존 자료 손실
- **B. /about 또는 /investors 페이지로 이전** — 권장. 메인은 영업 인입에 집중, 투자자/언론은 /about에서 보도록
- **C. 푸터에 토글 링크 — "기술 & 투자 정보" — 클릭 시 노출** — 어색함

**추천**: **옵션 B**. 기존 콘텐츠를 `/about`로 옮기고, 메인은 영업 인입 전용으로 재구성.
