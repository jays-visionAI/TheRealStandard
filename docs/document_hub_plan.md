# TRS Internal Document & Knowledge Hub (기획안)

## 1. 개요
TRS 어드민 내에서 사내 운영 매뉴얼, 교육 자료, 공지사항 및 외부 링크(구글 문서, 유튜브 등)를 효율적으로 관리하고 공유하기 위한 통합 지식 창고(Knowledge Hub) 시스템입니다.

## 2. 핵심 가치
- **중앙 집중화**: 산재된 교육 자료와 링크를 하나의 게시판에서 관리
- **확장성**: 탭 기반의 카테고리 시스템으로 자유로운 주제 분류
- **임베딩 최적화**: 구글 문서, 스프레드시트, 유튜브, 외부 웹페이지를 에디터 내에 직접 삽입

## 3. 주요 기능 정보

### A. 카테고리 관리 (Tab System)
- 사용자가 자유롭게 카테고리를 생성/수정/삭제 가능
- 상단 탭 형태로 구성하여 카테고리 간 빠른 전환 지원
- 예: [전체], [운영 매뉴얼], [신입사원 교육], [영업 자료], [물류 지침]

### B. 스마트 에디터 & 게시판
- **서식 편집**: 제목, 본문, 강조색, 리스트 등 기본 텍스트 서식 지원
- **외부 링크 임베딩 (Magic Link)**:
    - 구글 문서/시트: 공유 URL 입력 시 Iframe으로 즉시 렌더링
    - 유튜브: 영상 주소 입력 시 플레이어 임베딩
    - 웹사이트: 북마크 형태 또는 미리보기 카드로 표시
- **미리보기**: 작성 중인 문서의 실시간 렌더링 확인

### C. 문서 리조트
- 최근 수정 순서대로 리스트 표시
- 제목 및 태그 기반 검색 기능

## 4. UI/UX 디자인 방향
- **Theme**: TRS 어드민의 프리미엄 라이트 테마 유지
- **Layout**: 
    - 좌측/상단: 카테고리 탭
    - 우측/중앙: 문서 리스트 및 상세 보기
- **Interaction**: 
    - 문서 카드 호버 시 부드러운 확대 효과
    - 에디터 모드와 보기 모드의 매끄러운 전환

## 5. 기대 효과
- 신입 사원 온보딩 속도 개선
- 부서 간 지식 격차 해소 및 업무 표준화 강화
- 외부 툴(Google Workspace)과 TRS 시스템의 매끄러운 연동
